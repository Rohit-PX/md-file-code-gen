# md-file-code-gen
Library to parse go/shell script from md files

How ro run:
1. Clone the repo: https://github.com/Rohit-PX/md-file-code-gen/
2. Build the binary:
   ```
       go build -o validatedoc
   ```
3. Run the tool:
   ```
       ./validatedoc --mdfile <path-to-your-md-file> --kubeconfig <path-to-your-kubeconfig-file>
   ```
